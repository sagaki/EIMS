### 项目功能点
1.员工信息的增删改查
2.可适用邮箱进行账号密码的重置
3.被删除的数据可回档
### 项目优缺点
优点：项目结构简单易于理解<br>
缺点：由于是纯后端项目，未使用前端框架，界面不够美观。高耦合使得项目可扩展性、维护性较差。
### 项目参考资料

### 项目可优化点
使用前端框架，做前后端分离
